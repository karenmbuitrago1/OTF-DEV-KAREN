## Para empezar

Se debe tener instalado gulp en el ordenador, para poder visualizar localmente.

```
$ npm install gulp -g
```

### Dependencias

Se debe instalar las dependencias del proyecto.

```
$ npm install
```
npm install --save-dev gulp

## Compilar

Teniendo todo instalado, ahora debe compilar.

```
$ gulp
```
